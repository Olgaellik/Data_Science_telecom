# Анализ оттока клиентов телеком-компании

Этот проект основан на исследовании данных о клиентах телеком-оператора и выявлении факторов, влияющих на **отток пользователей**.  

## Цели

- Исследовать поведение клиентов и выявить ключевые факторы, влияющие на отток.  
- Сравнить активность и удержание клиентов в зависимости от типа услуг.  
- Построить простую модель, прогнозирующую, уйдёт ли клиент.  
- Визуализировать основные закономерности и представить результаты анализа в понятной форме.


## Используемые технологии

- **Python**
  - pandas, numpy — работа с данными  
  - matplotlib, seaborn — визуализация  
  - scikit-learn — модели машинного обучения  
- **Jupyter Notebook** — основной инструмент анализа

## Основные результаты

- Определены факторы, влияющие на отток клиентов (например, тип подключения, срок контракта, количество услуг).
- Визуализировано распределение признаков и различия между клиентами, которые остаются и уходят.
- Рассчитаны ключевые метрики и проведено сравнение групп клиентов.
- Построена базовая модель классификации для прогнозирования оттока.
- Сделаны выводы и предложены направления для улучшения удержания пользователей.
