# Телекоммуникации: предсказание оттока клиентов

**Описание проекта**: Оператор связи хочет бороться с оттоком клиентов. Для этого его сотрудники начнут предлагать промокоды и специальные условия всем, кто планирует отказаться от услуг связи. Чтобы заранее находить таких пользователей нужна модель, которая будет предсказывать, разорвёт ли абонент договор.

**Цель исследования:** Провести анализ данных клиентов и обучить модель, которая предскажет, кто из клиентов собираются завершить контракт.
